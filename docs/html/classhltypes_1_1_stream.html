bool hltypes::Stream::setCapacity ( int  newCapacity)

Resizes internal buffer.

Parameters
[in]newCapacityNew capacity of the internal buffer.
Returns
True if internal buffer was resized or already the same size that was requested.
Note
This does not change the data stream size. Use this to avoid allocation of too much data if not needed.
If newCapacity is smaller than the stream size, data will be lost and the Stream will be resized.
